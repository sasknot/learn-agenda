# Learn: Agenda

Built with [Vue 3](https://vuejs.org/) + [Typescript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/) + [Tailwind](https://tailwindcss.com/)

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
```

Files will be at `dist` folder
